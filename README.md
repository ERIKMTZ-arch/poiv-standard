# POI-V Standard
## Public Oriented Informative Infrastructure Vector
### Regenerative Cognitive Infrastructure for the territory Computing Era

> *"No AI system has territorial legitimacy if it degrades the material
> conditions that make life possible."*
> — POI-V Manifesto, Foundational Version 1.0, 2026

---

## What is POI-V?

POI-V is the first open certification framework requiring computational
infrastructure to generate net positive territorial regeneration.

Unlike existing standards (LEED, BREEAM, ASHRAE 90.4, Living Building
Challenge), POI-V does not aim to minimize negative impact.

It mandates that the presence of a computational node must measurably
improve the thermodynamic, hydrological, and ecological resilience
of its territory.

POI-V simultaneously defines a new architectural movement:
**Regional Metabolic Architecture**

Foundational axiom:
> *"Form follows metabolism."* — iarq / POI-V, 2026

---

## Core Principle

A data processing node has no territorial or urban legitimacy if it
operates in isolation from the vital cycles of its environment.

Every advanced industrial-scale computing system must function as a
decentralized public infrastructure that returns to the territory a
material, thermodynamic, and ecological capacity quantitatively and
qualitatively superior to what it extracts.

## The Bioregional Compatibility Principle

Not every territory is compatible with intensive cognitive infrastructure.

POI-V defines the Territorial Biophysical Limit (LBT):
the maximum sustainable computational load before net ecosystem degradation.


Any infrastructure exceeding its territory's ICDT is rejected under
POI-V — regardless of internal efficiency metrics.

---

## Key Formula: ECE replaces PUE


If you dissipate heat to the atmosphere instead of reusing it:
your infrastructure fails certification.
